Mulai Cepat

Ikuti 3 langkah berikut untuk mulai menggunakan Trello MCP:

  1. Dapatkan API Key & Token — Buat Power-Up Trello dan generate kredensial.
  2. Setup MCP Server — Install dan konfigurasi @delorenj/mcp-server-trello.
  3. Integrasi dengan OpenCode — Tambahkan MCP server ke opencode.json.

Tentang Trello MCP

MCP (Model Context Protocol) adalah standar terbuka yang memungkinkan AI Agent berkomunikasi dengan tools eksternal. Trello MCP Server (delorenj/mcp-server-trello) menyediakan jembatan antara Trello API dan AI Agent — sehingga agent bisa membaca, membuat, dan mengupdate kartu Trello langsung dari percakapan.

Server ini menangani rate limiting (300 req/10s per API key, 100 req/10s per token), validasi input, dan error handling secara otomatis.
