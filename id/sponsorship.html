Dokumentasi / Sponsor

Sponsorship — Dukung KAEDE

KAEDE adalah Trello Power-Up + MCP Server + CLI Orchestrator yang menjembatani Trello dengan AI Agent. Dibangun oleh Sandiko untuk PT Koneksi Jaringan Indonesia.

Dokumen ini transparan: setiap rupiah yang masuk tercatat di sini, setiap fitur yang dirilis terhubung ke target pendanaan.

Kenapa Mendukung KAEDE?

Untuk Tim di Indonesia

  • Bahasa Indonesia first — dokumentasi, CLI, playbook, semuanya
  • Gratis untuk internal — lisensi proprietary, tanpa biaya per-seat
  • Trello-based — tim sudah pakai Trello, tinggal pasang Power-Up

Untuk Ekosistem AI Agent

  • 42 tools (kaede-trello) + 45+ tools (upstream delorenj) — akses Trello paling lengkap dari AI Agent
  • Intent-driven — bilang "Mulai Sprint Alpha", AI Agent yang kerja
  • Playbook enforcement — SOP di Markdown, AI Agent yang jalankan

Untuk Open Source

  • 3 PR upstream ke delorenj/mcp-server-trello (fitur baru untuk semua)
  • CONTRIBUTING.md — panduan nambah tools baru, siapa pun bisa
  • Dual MCP architecture — mcp.kaede + mcp.trello, clean separation

FAQ

Q: Saya tidak punya uang, bagaimana cara bantu?
A: Banyak. Share ke tim lo, bikin tutorial, report bug, kontribusi kode — semua itu berharga.

Q: Apakah fitur akan di-lock untuk sponsor?
A: Tidak. Semua fitur tetap gratis untuk penggunaan internal. Reward sponsor bersifat seremonial & prioritas — bukan paywall.

Q: Dana dipakai untuk apa tepatnya?
A: Domain, hosting, produksi konten, dan insentif kontributor. Detail transaksi di atas.

Q: Kalau target dana tercapai, apa berhenti?
A: Target berikutnya langsung dibuka. Ini proyek jangka panjang.
