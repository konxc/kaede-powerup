# Sponsorship — Support KAEDE

KAEDE is a **Trello Power-Up + MCP Server + CLI Orchestrator** that bridges Trello with AI Agents. Built by Sandiko for PT Koneksi Jaringan Indonesia.

**This document is transparent:** every donation recorded here, every feature release tied to funding targets.

---

## Why Support KAEDE?

### For Teams
- **Free for internal use** — proprietary license, no per-seat cost
- **Trello-based** — your team already uses Trello, just add the Power-Up
- **Playbook enforcement** — SOPs in Markdown, AI Agent executes them

### For AI Agent Ecosystem
- **44 MCP tools** — most comprehensive Trello access from AI Agent
- **Intent-driven** — just say "Start Sprint Alpha", AI Agent does the work
- **Dual MCP architecture** — `mcp.kaede` + `mcp.trello`, clean separation

### For Open Source
- **3 upstream PRs** to `delorenj/mcp-server-trello` (new features for everyone)
- **CONTRIBUTING.md** — guide to add new tools, anyone can contribute
- **Active ecosystem contributor** — features first, upstream PRs later
